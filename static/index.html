Каждый синтез звучит чуть иначе — как живой диктор. 0 = одинаково каждый раз
Низкие значения (0.1-0.3) — ровная, предсказуемая речь. Высокие (0.7-1.0) — живее, но могут быть артефакты
Сколько вариантов рассматривает модель. Меньше = стабильнее, больше = разнообразнее
